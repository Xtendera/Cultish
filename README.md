# Cultish

This is a Fabric 1.21.10 mod that adds a new mob: the Prophet. Use the summoning ritual to summon him, and then run for cover!
Watch the demo for more info.

## Build
You should be able to use IntelliJ or Eclipse and build this project using gradle.
